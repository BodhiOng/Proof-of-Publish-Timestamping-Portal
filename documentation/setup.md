# System Setup Guide

This project is a Next.js dApp-style application with a Hardhat + Solidity backend for smart contracts.

## 1. Prerequisites

- Node.js 18+
- npm 9+
- MetaMask (for wallet flows)

## 2. Install Dependencies

From the project root:

```bash
npm install
```

## 3. Run the Web App

```bash
npm run dev
```

Open http://localhost:3000
